# Ali Hassan — Personal Portfolio

A modern, animated personal portfolio website built with React 19 and Vite, featuring smooth animations powered by Framer Motion. Showcasing projects, skills, and professional experience as a Backend Developer and Software Engineer.

🌐 **Live Demo:** [alihassan-tau.vercel.app](https://alihassan-tau.vercel.app)

---

## Tech Stack

| Technology | Version | Purpose |
|---|---|---|
| React | ^19.2.0 | UI framework |
| Vite | ^8.0.0-beta | Build tool & dev server |
| Framer Motion | ^12.34.0 | Animations & transitions |
| ESLint | ^9.39.1 | Code linting |

## Features

- **Smooth animations** — page transitions and element reveals powered by Framer Motion
- **Responsive design** — fully optimized for desktop, tablet, and mobile
- **Fast performance** — built with Vite for lightning-fast HMR and optimized production builds
- **Projects showcase** — Whisperfy, Promptly, Smart Video Insight and more
- **Skills section** — Python, Django, React.js, Next.js, REST APIs, NLP, and AI/ML
- **Contact section** — direct links to email, GitHub, and LinkedIn

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/ali-hassan1590/my-portfolio.git
cd my-portfolio

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

The app will be running at `http://localhost:5173`

## Available Scripts

# Start development server with HMR

## Deployment

This project is deployed on **Vercel**. To deploy your own:

```bash
# 1. Install Vercel CLI
npm install -g vercel

# 2. Build the project
npm run build

# 3. Deploy
vercel --prod
```

Or connect your GitHub repo directly to [vercel.com](https://vercel.com) for automatic deployments on every push.
